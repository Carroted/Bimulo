What's open-source?

Open-source means that anyone can view and edit the code that makes up Simulo. This means that anyone can contribute to the project and make it better for everyone!

Note: Simulo is currently in Alpha

Please note that Simulo is currently in Alpha, which means that it is not yet complete. Treat it not as a finished game/app, but instead as an engine demo.

We are building a custom game engine for Simulo, so treat this only as a test of that engine.

Finally, if you are on Firefox, please use a Chromium browser for this game! Firefox is not yet supported and is extremely slow.
